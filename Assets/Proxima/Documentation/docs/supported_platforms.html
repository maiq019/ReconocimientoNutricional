Supported Platforms

Proxima Inspector works by running an HTTP and WebSocket server in your game, which you can connect to from a web browser.

Proxima has been tested with different Unity veresions on the following platforms. See the test matrix below.

  • Windows (Win32 and UWP)
  • OSX
  • Android (+Meta Quest)
  • iOS
